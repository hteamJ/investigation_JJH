수사서류 작성 지원 도구

이 도구는 API를 호출하지 않습니다. 자료를 넣고 프롬프트를 복사해서, 원하는 AI(Gemini, ChatGPT, Claude.ai 등)에 붙여넣어 결과를 받는 방식입니다.
Claude에게 물어보러 가기 ↗ Gemini에게 물어보러 가기 ↗
먼저 "프롬프트 복사" 버튼으로 복사한 다음, 이 버튼을 눌러 새 탭에서 붙여넣기(Ctrl+V) 하시면 됩니다. 작업 중인 사건은 "사건 저장"으로 내려받아두면 나중에 "사건 불러오기"로 이어서 작업할 수 있어요.

개인정보 마스킹 (프롬프트를 복사할 때만 적용됩니다 — 화면에 입력된 원본은 그대로 유지)

추가 항목 (사건마다 다르게 필요한 항목을 직접 추가하세요 — 나.다.라. 등으로 이어집니다)
불특정형 사건처럼 "나. 쟁점" 구조 자체가 필요 없는 경우, 여기서 "나. 고소인 주장"처럼 직접 항목을 만들어 쓰세요. 내용을 비워두면, "수사결과보고서 작성" 시 AI가 원자료를 직접 확인해서 있는 내용은 자동으로 채워 넣고, 원자료에도 정말 없을 때만 "작성불가(이유: 정보부족)"로 표시합니다.
가. 피의자의 자백
비워두어도 괜찮습니다. 아래 "AI에게 전체 보고서 물어보기"를 쓰면, AI가 위 피의자신문조서 원문을 확인해서 자백 내용을 정리해줍니다.
나. 자백에 대한 보강증거
가. 다툼이 없이 인정되는 사실
⚠ 이 항목은 사건의 사실관계 토대가 되는 만큼, AI 초안을 그대로 쓰기보다 직접 작성하거나, 초안을 받더라도 원자료와 한 줄씩 대조 검증하는 것을 권장합니다.
나. 쟁점
아래 프롬프트로 AI에게 쟁점을 물어본 뒤, 답변 전체를 아래 칸에 붙여넣고 "자동으로 쟁점 카드 나누기"를 누르면 AI가 만든 형식(나./다./1)2)3)4))에 맞춰 카드가 자동으로 만들어져요. 형식이 안 맞으면 "쟁점 추가"로 직접 만드세요.
판례 추가 (쟁점을 먼저 정한 뒤, 각 쟁점에 맞는 판례를 찾아 여기 등록하세요)
슈퍼로이어 등에서 찾은 판례를 판례번호+내용 형태로 붙여넣으세요. 등록 후 위 쟁점 카드의 "3) 관련 판례" 칸에 옮겨 적거나, 아래 "판례 관련성 확인"으로 대조해보세요.
⚠ AI가 제시하는 판례번호는 실제로 존재하는 다른 사건 번호를 갖다 붙이거나, 번호 자체를 지어내는 경우가 있어요. AI 답변에 나온 판례는 그대로 쓰지 마시고, 반드시 casenote.kr, 국가법령정보센터, 종합법률정보 등에서 판례번호와 실제 판시내용이 일치하는지 직접 검색해서 확인한 뒤 판례 자료함에 등록해주세요.
